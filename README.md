# 🎨 "Real Tea" Product Packaging Design

Welcome to my graphic design portfolio! This repository showcases a modern, eco-friendly product packaging concept created using **Canva**.

## ✨ Design Highlights
* **Theme:** Rustic kraft paper texture for a natural, organic look.
* **Typography:** Bold, expressive brush-script font for the main branding.
* **Graphics:** Minimalist green leaf illustration and clean badge icons.
* **Tools Used:** Canva

This concept is designed to give a premium, sustainable feel to a beverage brand.
